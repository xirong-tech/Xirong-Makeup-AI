# 歖容-全龄美妆智库 AI 小程序

## 🌟 项目核心亮点
1. **双模型AI妆容迁移**：融合PSGAN+&BeautyGAN快速高效模型、MAD跨域扩散模型高精度模型，适配不同妆容效果需求，针对亚洲女性面部特征优化，鲁棒性更强。
2. **定制化亚洲人脸数据集**：基于CelebA_HQ/MT-Dataset开源数据集优化，补充国内女性不同光照/脸型/肤质/妆容数据，构建妆容推荐/迁移/教学三位一体专属数据集。
3. **全流程美妆智能服务**：覆盖「个性化妆容推荐-肤质检测报告-化妆品精准推荐-用户社区交流」四大核心功能，实现从技术到产品的完整落地。
4. **轻量化微信小程序载体**：依托微信小程序开发，无需额外下载，主页下拉即可使用，适配碎片化学习与美妆操作需求。
5. **技术全开源可验证**：所有AI算法、小程序代码、数据集小样均开源，配套详细技术文档与测试报告，技术落地性全程可追溯。
6. **冷启动推荐优化**：融合MAMEX冷启动推荐算法，解决用户初期使用阶段妆容/产品推荐有效性低的问题，适配全龄段用户。
